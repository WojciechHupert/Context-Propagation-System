System Roadmap

8 Total Phases
87.5% COMPLETE
01 Action Records
COMPLETED +

Implementation of the ActionRecords schema. Automated LLM extraction of structured future intents, relationship shifts, and narrative commitments from live dialogue. Features dynamic retrieval injection into the prompt context based on trigger windows and cooldowns.

02 Canonical Propagation
COMPLETED +

Deployment of the PropagationEvents architecture. Information spreads via associative routing, prioritizing Subjects within shared social circles. Each transfer generates a distinct recipient-owned record while maintaining a traceable link to its origin.

03 Continuity Subsystem
STABILIZED +

Integration of the UMoiraiContinuitySubsystem within Unreal Engine. Enables temporal chaining of memories using PreviousMemoryId to reconstruct long-term narrative threads. Includes mutation tracking and drift counters to monitor cognitive evolution over time.

04 Moirai Studio Pro
STABILIZED +

Development of an IDE-style administrative suite. Provides real-time neural vitals, flux tracking, and manual moderation of transcripts and memories. Features a master-detail layout for deep-diving into canonical memory writes and chaining metadata.

4.5 Runtime Hardening
COMPLETED +

Critical safety layer implementation. Includes assistant dialogue sanitization before persistence and transcript authority filtering to exclude game_fallback or error rows. Introduces a one-step repair pass for malformed LLM outputs to maintain context hygiene.

06 Human Recall
COMPLETED +

Advanced cognitive filtering. Implements retrieval bucketing (Recent, Stable, Social, Emotional) with strict token caps. Features a human-centric recall layer where memories exhibit age-based vagueness and semantic blurring based on their RecallStability rating.

07 Studio Observability
COMPLETED +

Enhanced diagnostic tooling. Includes privacy level enforcement (shareable vs. private) and source attribution badges. Features propagation trace visualization and a refined inspector UI exposing 11 distinct metadata fields per memory record for deep auditability.

08 World Reaction Layer
PLANNED +

Bridging the gap between cognition and environment. Enables direct coupling between Moirai continuity states and the world simulation. Memories now drive environmental shifts and Subject behavior changes based on historical persistence.
